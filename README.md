**Calculator**
A responsive, web-based calculator built with HTML, CSS, and JavaScript.
This project demonstrates clean UI design, modular code structure, and practical application of front‑end development skills.

**Features**
-Perform basic arithmetic operations: addition, subtraction, multiplication, division
-Clear and minimal user interface
-Responsive layout for different screen sizes
-Easy to extend with new functions

**Project Structure**
Calculator/
│
├── index.html        → Main HTML file
├── calculator.css    → Stylesheet for UI
└── calculator.js     → JavaScript logic

**Getting Started**
-Clone the repository:
  git clone https://github.com/pallavipawaskar/Calculator.git
-Navigate into the project folder:
  cd Calculator
-Open index.html in your browser to start using the calculator.
